# Whispering Death

<img src="/assets/images/toothless/kb-img-whispering-death.png" width="120" style="float: right; margin-left: 1rem;">

The Whispering Death is a series canon dragon, first appearing in Riders & Defenders of Berk. It is an Archipelago Additions dragon on theDeadly Nadderbase.

## Stats

| Stat | Value |
|---|---|
| Health | 60 (30 hearts) |
| Armour | 5 (2.5 icons) |
| Attack | 8 (4 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.15 (50% faster than Allay speed) |

## Taming

**Taming Foods:** Raw Cod

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Whispering Death Scales, 0-1 Whispering Death Spines

**Death Drops:** 1-2 Dragon Blood, 3-5 Whispering Death Scales, 1-2 Whispering Death Spines

## Spawn Biomes

- `minecraft:dripstone_caves`
- `minecraft:lush_caves`
- `minecraft:jagged_peaks`
- `minecraft:stony_peaks`
- `minecraft:windswept_gravelly_hills`
