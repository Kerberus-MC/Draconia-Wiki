# Screaming Death

<img src="/assets/images/toothless/kb-img-screaming-death.png" width="120" style="float: right; margin-left: 1rem;">

The Screaming Death is a series canon dragon, first appearing in Defenders of Berk. It is an Archipelago Additions dragon on theDeadly Nadderbase.

## Stats

| Stat | Value |
|---|---|
| Health | 500 (250 hearts) |
| Armour | 10 (5 icons) |
| Attack | 23 (11.5 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (50% faster than Allay speed) |

## Taming

**Taming Foods:** Large Feed Bundle

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Unbreedable

## Drops

**Brush Drops:** 3-5 Screaming Death Scales, 0-1 Screaming Death Spines

**Death Drops:** 10-20 Dragon Blood, 7-15 Screaming Death Scales, 1-2 Screaming Death Spines

## Spawn Biomes

- `This dragon only spawns in the Whispering Death Nest structure with the Wild Aggression Effect.`
- `Screaming Death Eggs can be found from mining Whispering Death Nest Blocks.`
