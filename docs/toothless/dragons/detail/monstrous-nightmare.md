# Monstrous Nightmare

<img src="/assets/images/toothless/kb-img-monstrous-nightmare.png" width="120" style="float: right; margin-left: 1rem;">

The Monstrous Nightmare is a main canon dragon from the first film. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 120 (60 hearts) |
| Armour | 4 (2 icons) |
| Attack | 7 (3.5 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (50% faster than an Allay) |

## Taming

**Taming Foods:** Raw Pork, Raw Mutton

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held
No armour worn
Fire resistance potion effect active

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious stew, Sagefruit

## Drops

**Brush Drops:** 2-3 Monstrous Nightmare Scales, 1-2 Monstrous Nightmare Gel

**Death Drops:** 1-3 Dragon Blood, 4-7 Monstrous Nightmare Scales, 1-3 Monstrous Nightmare Gel

## Spawn Biomes

- `minecraft:stony_peaks`
- `minecraft:windswept_hills`
- `minecraft:windswept_gravelly_hills`
