# Light Fury

<img src="/assets/images/toothless/kb-img-light-fury.png" width="120" style="float: right; margin-left: 1rem;">

The Light Fury is a main canon dragon from the third film. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 60 (30 hearts) |
| Armour | 0 |
| Attack | 2 (1 heart) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.2 (twice Allay speed) |

## Taming

**Taming Foods:** Raw Cod, Raw Salmon

**Requirements:** Invisibility or Night Vision potion effect active
No armour equipped
No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Honeycomb, Sagefruit

## Drops

**Brush Drops:** 1-3 Light Fury Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Light Fury Scales

## Spawn Biomes

- `minecraft:birch_forest`
- `minecraft:old_growth_birch_forest`
