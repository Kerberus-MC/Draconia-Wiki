# Triple Stryke

<img src="/assets/images/toothless/kb-img-triple-stryke.png" width="120" style="float: right; margin-left: 1rem;">

The Triple Stryke is a series canon dragon, first appearing in Race to the Edge. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 90 (45 hearts) |
| Armour | 2 (1 icon) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.4 |
| Fly Speed | 0.1 |

## Taming

**Taming Foods:** Raw Beef

**Requirements:** Decrease dragon below 25% hp

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Triple Stryke Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Triple Stryke Scales, 2-3 Venom Gland

## Spawn Biomes

- `minecraft:desert`
- `minecraft:badlands`
- `minecraft:wooded_badlands`
