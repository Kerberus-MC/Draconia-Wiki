# Butterflywing

<img src="/assets/images/toothless/kb-img-butterflywing.png" width="120" style="float: right; margin-left: 1rem;">

The Butterflywing is a dragon from the China-exclusive game. It is an Archipelago Additions dragon on theTerrible Terrorbase.

## Stats

| Stat | Value |
|---|---|
| Health | 40 (20 hearts) |
| Armour | 0 |
| Attack | 4 (2 hearts) |
| Walk Speed | 0.3 (Wolf speed) |
| Fly Speed | 0.06 (just over half Allay speed) |

## Taming

**Taming Foods:** Any flower item

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Pufferfish, Sagefruit

## Drops

**Brush Drops:** 1-2 Butterflywing Scales

**Death Drops:** 1 Dragon Blood, 1-3 Butterflywing Scales

## Spawn Biomes

- `minecraft:flower_forest`
- `minecraft:meadow`
- `minecraft:sunflower_plains`
