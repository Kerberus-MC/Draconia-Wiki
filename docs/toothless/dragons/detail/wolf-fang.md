# Wolf-Fang

<img src="/assets/images/toothless/kb-img-wolf-fang.png" width="120" style="float: right; margin-left: 1rem;">

The Wolf-Fang is a book canon dragon. It is an Archipelago Additions dragon on theSpeed Stingerbase.

## Stats

| Stat | Value |
|---|---|
| Health | 55 (27.5 hearts) |
| Armour | 2 (1 icon) |
| Attack | 3 (1.5 hearts) |
| Walk Speed | 0.3 (Wolf speed) |

## Taming

**Taming Foods:** N/A

**Requirements:** This dragon is not tameable.

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit
