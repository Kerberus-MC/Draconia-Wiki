# Deadly Nadder

<img src="/assets/images/toothless/kb-img-deadly-nadder.png" width="120" style="float: right; margin-left: 1rem;">

The Deadly Nadder is a main canon dragon from the first film. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 50 (25 hearts) |
| Armour | 2 (1 icon) |
| Attack | 8 (4 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (50% faster than an Allay) |

## Taming

**Taming Foods:** Raw Chicken

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Deadly Nadder Scales, 0-1 Deadly Nadder Spine

**Death Drops:** 1-2 Dragon Blood, 3-5 Deadly Nadder Scales, 1-2 Deadly Nadder Spine

## Spawn Biomes

- `minecraft:bamboo_jungle`
- `minecraft:birch_forest`
- `minecraft:jungle`
- `minecraft:old_growth_birch_forest`
- `minecraft:sparse_jungle`
- `minecraft:river`
