# Windwalker

<img src="/assets/images/toothless/kb-img-windwalker.png" width="120" style="float: right; margin-left: 1rem;">

The Windwalker is a game canon dragon. It is an Archipelago Additions dragon on theSkrillbase.

## Stats

| Stat | Value |
|---|---|
| Health | 85 (42.5 hearts) |
| Armour | 2 (1 icon) |
| Attack | 4 (2 hearts) |
| Walk Speed | 0.2 (Cow speed) |
| Fly Speed | 0.2 (double Allay speed) |

## Taming

**Taming Foods:** Raw Cod, Raw Salmon, Tropical Fish

**Requirements:** Decrease dragon below 25% hp

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Windwalker Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Windwalker Scales

## Spawn Biomes

- `minecraft:beach`
- `minecraft:snowy_beach`
- `minecraft:stony_shore`
