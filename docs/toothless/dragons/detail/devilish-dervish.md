# Devilish Dervish

<img src="/assets/images/toothless/kb-img-devilish-dervish.png" width="120" style="float: right; margin-left: 1rem;">

The Devilish Dervish is a game canon dragon. It is a Stego's Dragons dragon, now part of AA after the merge. The Devilish Dervish uses theMonstrous Nightmarebase.

## Stats

| Stat | Value |
|---|---|
| Health | 100 (50 hearts) |
| Armour | 4 (2 icons) |
| Attack | 8 (4 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (40% faster than Allay speed) |

## Taming

**Taming Foods:** Raw Pork, Raw Mutton

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held
No armour worn
Fire resistance potion effect active

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Devilish Dervish Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Devilish Dervish Scales

## Spawn Biomes

- `minecraft:savanna_plateau`
- `minecraft:windswept_savanna`
- `minecraft:savanna`
