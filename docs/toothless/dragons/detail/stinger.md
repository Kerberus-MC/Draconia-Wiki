# Stinger

<img src="/assets/images/toothless/kb-img-stinger.png" width="120" style="float: right; margin-left: 1rem;">

The Stinger is a dubiously canon dragon from the Arena Spectacular. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 80 (40 hearts) |
| Armour | 2 (1 icon) |
| Attack | 8 (4 hearts) |
| Walk Speed | 0.55 (Evoker speed) |

## Taming

**Taming Foods:** Raw Mutton

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Stinger Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Stinger Scales

## Spawn Biomes

- `minecraft:meadow`
- `minecraft:plains`
- `minecraft:savanna`
- `minecraft:savanna_plateau`
- `minecraft:windswept_savanna`
