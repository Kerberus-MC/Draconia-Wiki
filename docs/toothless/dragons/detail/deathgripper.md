# Deathgripper

<img src="/assets/images/toothless/kb-img-deathgripper.png" width="120" style="float: right; margin-left: 1rem;">

The Deathgripper is a main canon dragon, first appearing in HTTYD 3. It is an Archipelago Additions dragon on theTriple Strykebase.

## Stats

| Stat | Value |
|---|---|
| Health | 70 (35 hearts) |
| Armour | 2 (1 icon) |
| Attack | 5 (2.5 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.17 (around twice Allay speed) |

## Taming

**Taming Foods:** Dragon Blood Bottle, Dragon Blood Marinated Feed

**Requirements:** Decrease dragon below 25% hp

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Deathgripper Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Deathgripper Scales, 1 Venom Gland

## Spawn Biomes

- `minecraft:bamboo_jungle`
- `minecraft:sparse_jungle`
- `minecraft:swamp`
