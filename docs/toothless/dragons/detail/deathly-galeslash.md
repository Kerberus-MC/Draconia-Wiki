# Deathly Galeslash

<img src="/assets/images/toothless/kb-img-deathly-galeslash.png" width="120" style="float: right; margin-left: 1rem;">

The Deathly Galeslash is a game canon hybrid of theDeadly Nadder&Stormcutter, first appearing in Titans Uprising. It is an Archipelago Additions dragon on theDeadly Nadderbase.

## Stats

| Stat | Value |
|---|---|
| Health | 80 (40 hearts) |
| Armour | 2 (1 icons) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 |

## Taming

**Taming Foods:** Raw Chicken, Raw Cod

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Deathly Galeslash Scales, 0-1 Deathly Galeslash Spines

**Death Drops:** 1-2 Dragon Blood, 3-5 Deathly Galeslash Scales, 1-2 Deathly Galeslash Spines

## Spawn Biomes

- `minecraft:ice_spikes`
- `minecraft:river`
- `minecraft:frozen_river`
