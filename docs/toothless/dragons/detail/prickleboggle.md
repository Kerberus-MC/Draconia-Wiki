# Prickleboggle

<img src="/assets/images/toothless/kb-img-prickleboggle.png" width="120" style="float: right; margin-left: 1rem;">

The Prickleboggle is a game canon dragon. It is a Stego's Dragons dragon, now part of AA after the merge. The Prickleboggle uses theMonstrous Nightmarebase.

## Stats

| Stat | Value |
|---|---|
| Health | 80 (40 hearts) |
| Armour | 7 (3.5 icons) |
| Attack | 3 (1.5 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (40% faster than Allay speed) |

## Taming

**Taming Foods:** Raw Cod

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held
No armour worn
Fire resistance potion effect active

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Prickleboggle Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Prickleboggle Scales

## Spawn Biomes

- `minecraft:grove`
- `minecraft:snowy_plains`
- `minecraft:snowy_slopes`
