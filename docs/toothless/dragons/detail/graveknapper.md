# Graveknapper

<img src="/assets/images/toothless/kb-img-graveknapper.png" width="120" style="float: right; margin-left: 1rem;">

The Graveknapper is a game canon hybrid of theBoneknapper&Screaming Death, first appearing in School of Dragons. It is an Archipelago Additions dragon on the Deadly Nadder base.

## Stats

| Stat | Value |
|---|---|
| Health | 300 (150 hearts) |
| Armour | 20 (10 icons) |
| Attack | 18 (9 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (50% faster than Allay speed) |

## Taming

**Taming Foods:** Bone Feed Bundle

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Unbreedable

## Drops

**Brush Drops:** 3-5 Graveknapper Scales, 0-1 Graveknapper Spines

**Death Drops:** 10-20 Dragon Blood, 7-15 Graveknapper Scales, 1-2 Graveknapper Spines

## Spawn Biomes

- `This dragon only spawns in the Whispering Death Nest structure with the Wild Aggression Effect.`
- `Graveknapper Eggs can be found from mining Whispering Death Nest Blocks.`
