# Scuttleclaw

<img src="/assets/images/toothless/kb-img-scuttleclaw.png" width="120" style="float: right; margin-left: 1rem;">

The Scuttleclaw is a main canon dragon, first appearing in HTTYD 2. It is an Archipelago Additions dragon on theGroncklebase.

## Stats

| Stat | Value |
|---|---|
| Health | 50 (25 hearts) |
| Armour | 0 |
| Attack | 3 (1.5 hearts) |
| Walk Speed | 0.4 |
| Fly Speed | 0.14 |

## Taming

**Taming Foods:** Raw Cod, Raw Salmon

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Scuttleclaw Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Scuttleclaw Scales

## Spawn Biomes

- `minecraft:grove`
- `minecraft:snowy_plains`
- `minecraft:snowy_slopes`
- `minecraft:snowy_beach`
