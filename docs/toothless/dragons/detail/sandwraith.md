# Sandwraith

<img src="/assets/images/toothless/kb-img-sandwraith.png" width="120" style="float: right; margin-left: 1rem;">

The Sandwraith is a game canon dragon. It is an Archipelago Additions dragon on theLight Furybase.

## Stats

| Stat | Value |
|---|---|
| Health | 60 (30 hearts) |
| Armour | 0 |
| Attack | 2 (1 heart) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.14 (around 50% faster than an Allay) |

## Taming

**Taming Foods:** Tropical Fish

**Requirements:** Invisibility or Night Vision potion effect active
No armour equipped
No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Honeycomb, Sagefruit

## Drops

**Brush Drops:** 1-3 Sandwraith Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Sandwraith Scales

## Spawn Biomes

- `minecraft:beach`
- `minecraft:desert`
- `minecraft:snowy_beach`
