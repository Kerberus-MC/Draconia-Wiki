# Gronckle

<img src="/assets/images/toothless/kb-img-gronckle.png" width="120" style="float: right; margin-left: 1rem;">

The Gronckle is a main canon dragon from the first film. It is a base Isle of Berk dragon.

## Stats

| Stat | Value |
|---|---|
| Health | 60 (30 hearts) |
| Armour | 10 (5 icons) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.37 (Piglin Brute speed) |
| Fly Speed | 0.08 (just below Allay speed) |

## Taming

**Taming Foods:** Raw Beef

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Gronckle Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Gronckle Scales

## Spawn Biomes

- `minecraft:meadow`
- `minecraft:plains`
- `minecraft:savanna`
- `minecraft:savanna_plateau`
- `minecraft:stony_shores`
- `minecraft:stony_peaks`
