# Sentinel

<img src="/assets/images/toothless/kb-img-sentinel.png" width="120" style="float: right; margin-left: 1rem;">

The Sentinel is a series canon dragon, first appearing in Race to the Edge. It is a Archipelago Additions dragon on theMonstrous Nightmarebase.

## Stats

| Stat | Value |
|---|---|
| Health | 120 (60 hearts) |
| Armour | 13 (6.5 icons) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.35 (Vindicator speed) |
| Fly Speed | 0.1 (Allay speed) |

## Taming

**Taming Foods:** Vanaheim Fruit

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held
No armour worn
Fire resistance potion effect active

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 0-1 Cobblestone, 2-3 Sentinel Scales

**Death Drops:** 1-3 Cobblestone, 1-2 Dragon Blood, 6-9 Sentinel Scales

## Spawn Biomes

- `minecraft:lush_caves`
- `minecraft:jungle`
- `minecraft:sparse_jungle`
