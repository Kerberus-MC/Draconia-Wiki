# Shockjaw

<img src="/assets/images/toothless/kb-img-shockjaw.png" width="120" style="float: right; margin-left: 1rem;">

The Shockjaw is a game canon dragon. It is a Stego's Dragons dragon, now part of AA after the merge. The Shockjaw uses theSkrillbase.

## Stats

| Stat | Value |
|---|---|
| Health | 90 (45 hearts) |
| Armour | 2 (1 icon) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.4 (Piglin Brute speed) |
| Fly Speed | 0.19 (almost twice Allay speed) |

## Taming

**Taming Foods:** Raw Cod, Raw Salmon, Tropical Fish

**Requirements:** Decrease dragon below 25% hp

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Shockjaw Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Shockjaw Scales

## Spawn Biomes

- `minecraft:beach`
- `minecraft:snowy_beach`
- `minecraft:stony_shore`
