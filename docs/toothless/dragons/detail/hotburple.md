# Hotburple

<img src="/assets/images/toothless/kb-img-hotburple.png" width="120" style="float: right; margin-left: 1rem;">

The Hotburple is a main canon dragon, first appearing in HTTYD 2. It is an Archipelago Additions dragon on theGroncklebase.

## Stats

| Stat | Value |
|---|---|
| Health | 80 (40 hearts) |
| Armour | 10 (5 icon) |
| Attack | 6 (3 hearts) |
| Walk Speed | 0.3 |
| Fly Speed | 0.06 |

## Taming

**Taming Foods:** Raw Cod, Raw Iron

**Requirements:** No tools or weapons (except Dragon Blades & specified Viking Artifacts ) held

**Alt Taming:** Dragon Nip

## Breeding

**Breeding Foods:** Suspicious Stew, Sagefruit

## Drops

**Brush Drops:** 1-3 Hotburple Scales

**Death Drops:** 1-2 Dragon Blood, 3-5 Hotburple Scales

## Spawn Biomes

- `minecraft:forest`
- `minecraft:flower_forest`
- `minecraft:sunflower_plains`
